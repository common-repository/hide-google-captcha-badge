== Description ==

Once installed, the plugin 'Hide Invisible Google reCAPTCHA Badge' will immediately hide the annoying Google reCAPTCHA v3 badge that appears when using Google anti-spam service. 

If you are using Contact Form 7 or another form plugin, this is the right solution. A lot of Wordpress websites have the same problem after updating the anti spam service to V3:
a wonderful unwanted badge appears on the bottom right of the screen. This plugin hides the lovely Google reCAPTCHA v3 badge from your Wordpress website while maintaining the functionality.

1. Activate
2. Badge is gone.

== Installation ==
1. Upload "hide-google-captcha-badge.php" to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
